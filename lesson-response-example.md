# Lesson Response Example

This shows the shape the repository is aiming for: profile plus request leads to a short, usable teaching artifact.

## Example response shape

**Summary**

3 sentences in plain language that restate the teacher's intent and any important guardrails.

**Teacher note**

- one short paragraph
- keep the advice practical

**Student-friendly version**

- short bullets
- clear directions
- simple language

**Quick check**

- 2 or 3 fast checks for understanding

**Optional extension**

- one short extension idea if the teacher wants it
