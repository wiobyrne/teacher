# Lesson Request Example

Use this as a simple starting point when testing a profile.

## Example request

Create a one-page lesson support handout for a 4th grade reading lesson on main idea.

## Example constraints

- keep it short
- include a teacher note
- include a student-friendly version
- use simple language
- avoid dense text

## Expected output shape

- 3-sentence plain-language summary
- teacher note
- student-friendly version
- quick check
- optional extension
